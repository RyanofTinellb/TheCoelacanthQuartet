The Coelacanth Quartet

Eons ago, I decided to write a novel. I then decided, as a linguist, I should create a number of constructed languages, and now I've decided to put this on the web. So I'm two levels away from actually writing the thing. One day….

Although there is only one story, it appears here in five versions. On each page, there is the option to jump between these.

  • English

    • The original Coelacanth Quartet.

  • Tinellbian

    • Each chapter of the original text, translated into a different Tinellbian language, using the native scripts.

  • Transliteration

    • The same as above, but using Roman letters.

  • Interlinear

    • The meaning of each morpheme, using the glossing rules and abbreviations here.

  • Gloss

    • Just the meaning of each morpheme, lacking the native pronunciations. Boring, really, but might work as freeform poetry.
